# dio-trilha-go-problemas-numericos
Desafios de projeto da trilha formação em Go. Para ambos os desafios, deverão ser criados programas que trabalhem com % e for.

# Desafio 1
Criar um programa que exiba todos os números entre 1 e 100 que são divisíveis por 3.

# Desafio 2

Recriar em código uma brincadeira em que ao falar os numeros, sempre que aparecer um múltiplo de 3, o participante deve falar "Pin" e nos múltiplos de 5, o participante deve falar "Pan". Então, o programa deve imprimir os números de 1 a 100, e nos casos citados, não devem aparecer os números, mas sim, o que o participante deve falar.

Em caso de múltiplos de 3 e 5 ao mesmo tempo, o programa deve exibir a mensagem "Pin pan"
